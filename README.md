# Project Overview
This application is meant to varify and test the value of a credit card number and keep the number hidden with the exception of the last four digit number.

# Prerequisites
```shell
    NodeJS >= v16.16.0
    npm >= v8.11.0
    jest >= ^29.4.3
    Chrome/Firefox/Safari/Edge >= Latest major versions
```

# NodeJS test file

### to run this project
```Shell
    npm install
    npm start
```
### to test this project
```Shell
    npm i
    npm test
```
